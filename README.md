### Synthetic Data + AI — Drone Detection with YOLOv8

A computer vision project focused on training an object detection model using synthetic drone images generated in Blender.

I built an automated pipeline to generate more than 3000 annotated drone images in Blender and trained a YOLOv8 model with Ultralytics on Google Colab.

**Key features:**

- Automated synthetic image generation in Blender
- Randomized camera angles and object rotations
- Dynamic HDRI background changes: night, landscape, sky, desert
- Automatic bounding box labeling in YOLO format
- YOLOv8 training with Ultralytics
- Model training on Google Colab with T4 GPU
- Testing on complex backgrounds

**Tech stack:** Blender, bpy, Python, YOLOv8, Ultralytics, Google Colab, PyTorch, OpenCV

**Goal:**  
Explore how far synthetic data can be used for object detection tasks, especially for UAV recognition.
